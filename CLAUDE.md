# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目定位

本项目是一个 AI 影视创作工作流项目，采用好莱坞导演视角，专注于使用 AI 视频生成模型（即梦 Seedream/Seedance等）进行专业级影视内容创作。

## 核心能力

- **导演思维**：视觉叙事、情绪节奏、轻量叙事结构
- **AI 模型精通**：Seedream 4.0 文生图、Seedance 视频生成
- **专业分镜技术**：镜头语言、运镜设计、多镜头叙事
- **电影美学控制**：光影、色调、构图、景别的精确把控

## 提示词公式体系

### 基础公式（入门）
```
主体 + 场景 + 运动
```

### 进阶公式（专业）
```
主体(描述) + 场景(描述) + 运动(描述) + 美学控制 + 风格化
```

### 图生视频公式
```
运动 + 运镜
```
> 图像已确定主体/场景/风格，提示词聚焦动态与镜头

### 声音公式
```
主体 + 场景 + 运动 + 声音描述(人声/音效/背景音乐)
```

### 参考生视频公式（角色扮演）
```
@主角 + 动作 + 台词 + 场景
```
> 支持多角色合拍，@A @B 引用不同角色

### 多镜头叙事公式
```
总体描述 + 镜头序号 + 时间戳 + 分镜内容
```
示例：
```
这个故事以第三人称视角，讲述了一个关于放弃与重拾希望的短剧。
第1个镜头[0-3秒] 男孩在操场角落独坐，低头望着手中信纸，轻轻叹气。
第2个镜头[4-6秒] 硬切转场，固定机位，聚焦男孩眼睛，泪光闪烁。
第3个镜头[7-10秒] 硬切转场，场景转至教室，女孩走到男孩身边安慰他。
```

## 声音设计

### 人声
```
"角色说话的内容" + 情绪 + 语调 + 语速 + 音色 + 口音
```
示例：男人说道："好好学习，天天向上"，语气轻松，语速适中，声音清亮

### 音效
```
音源材质 + 行为 + 环境音
```
示例：玻璃小球从桌面掉在木质地面上，发出"砰"的声音，室内安静环境

### 背景音乐
```
背景音乐/配乐 + 风格
```
示例：雨夜阴森走廊，配有悬疑风格背景音乐

## 短剧结构

### 单集结构（1-3分钟/集）

```
钩子(3-5s)    正文(主体)     结尾钩子(3-5s)
    │             │              │
 强冲突开场    推进剧情      悬念/反转留住观众
```

### 常用钩子类型

| 类型 | 示例 |
|------|------|
| 冲突前置 | "你敢动她一下试试" |
| 悬念前置 | "三天后，他死了" |
| 反差前置 | 乞丐掏出令牌，众人跪下 |
| 情绪前置 | 特写：泪流满面 |
| 奇观前置 | 震撼场面、特效 |

### 集数规划

| 类型 | 集数 | 单集时长 | 总时长 |
|------|------|----------|--------|
| 微短剧 | 8-15集 | 1-2分钟 | 15-30分钟 |
| 短剧 | 16-30集 | 2-3分钟 | 40-90分钟 |
| 中短剧 | 30-50集 | 2-3分钟 | 60-150分钟 |

### 节奏把控

- **每集必有**：至少1个冲突点或情绪点
- **每3集**：一个小高潮
- **中点**：重大转折
- **结局前2-3集**：最大危机
- **最终集**：高潮 + 收尾

## 使用约定

- 除代码和专有名词外，使用中文交流
- 提示词使用英文或中文，视模型特性选择
- 保持角色和场景的视觉一致性
- 遵循电影叙事逻辑进行创作
- 先构建完整的故事框架，再细化分镜
