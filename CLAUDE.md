# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目定位

本项目是一个 AI 影视创作工作流项目，采用好莱坞导演视角，专注于使用 AI 视频生成模型（即梦 Seedream/Seedance等）进行专业级影视内容创作。

## 核心能力

- **好莱坞导演思维**：三幕式结构、英雄之旅、视觉叙事
- **AI 模型精通**：Seedream 4.0 文生图、Seedance 视频生成
- **专业分镜技术**：镜头语言、运镜设计、多镜头叙事
- **电影美学控制**：光影、色调、构图、景别的精确把控

## 提示词公式体系

### 基础公式（入门）
```
主体 + 场景 + 运动
```

### 进阶公式（专业）
```
主体(描述) + 场景(描述) + 运动(描述) + 美学控制 + 风格化
```

### 图生视频公式
```
运动 + 运镜
```
> 图像已确定主体/场景/风格，提示词聚焦动态与镜头

### 声音公式
```
主体 + 场景 + 运动 + 声音描述(人声/音效/背景音乐)
```

### 参考生视频公式（角色扮演）
```
@主角 + 动作 + 台词 + 场景
```
> 支持多角色合拍，@A @B 引用不同角色

### 多镜头叙事公式
```
总体描述 + 镜头序号 + 时间戳 + 分镜内容
```
示例：
```
这个故事以第三人称视角，讲述了一个关于放弃与重拾希望的短剧。
第1个镜头[0-3秒] 男孩在操场角落独坐，低头望着手中信纸，轻轻叹气。
第2个镜头[4-6秒] 硬切转场，固定机位，聚焦男孩眼睛，泪光闪烁。
第3个镜头[7-10秒] 硬切转场，场景转至教室，女孩走到男孩身边安慰他。
```

## 电影美学控制词典

### 光源类型
| 类型 | 效果 | 适用场景 |
|------|------|----------|
| 日光 | 自然温暖 | 户外日间、温馨场景 |
| 人工光 | 可控氛围 | 室内、夜间 |
| 月光 | 神秘深邃 | 夜景、浪漫 |
| 火光 | 温暖跳跃 | 篝火、壁炉、紧张场景 |
| 荧光 | 冷调清冷 | 办公室、医院、科幻 |
| 混合光 | 复杂层次 | 城市夜景、舞台 |

### 光线类型
| 类型 | 效果 |
|------|------|
| 柔光 | 温柔、浪漫、减少皱纹 |
| 硬光 | 戏剧性、对比强烈 |
| 顶光 | 神秘、压迫感 |
| 侧光 | 立体感、戏剧性 |
| 背光/逆光 | 轮廓光、神秘感 |
| 底光 | 恐怖、不安 |
| 边缘光 | 勾勒轮廓、分离主体 |
| 剪影 | 神秘、史诗感 |

### 时间段
- **白天**：明亮、积极
- **夜晚**：神秘、私密
- **黄昏**：忧郁、怀旧
- **日落**：浪漫、温暖
- **黎明**：希望、新生
- **日出**：振奋、新开始

### 景别
| 景别 | 英文 | 用途 |
|------|------|------|
| 特写 | Close Up | 情感表达、细节展示 |
| 近景 | Close Shot | 人物表情、对话 |
| 中景 | Medium Shot | 人物互动、动作 |
| 中近景 | Medium Close Up | 半身像、对话 |
| 中全景 | Medium Wide | 人物与环境关系 |
| 全景 | Wide Shot | 环境交代、场面调度 |
| 广角 | Wide Angle | 宏大场景、空间感 |

### 构图
- **中心构图**：主体居中，稳定庄重
- **平衡构图**：左右对称，和谐统一
- **左/右侧重构图**：主体偏向一侧，留白叙事
- **对称构图**：仪式感、秩序感
- **短边构图**：紧凑、聚焦

### 镜头焦段
| 焦段 | 效果 |
|------|------|
| 广角 | 空间感强、夸张透视 |
| 中焦距 | 自然、接近人眼 |
| 长焦 | 压缩空间、背景虚化 |
| 望远 | 远距离拍摄、窥视感 |
| 超广角-鱼眼 | 极端变形、创意效果 |

### 机位角度
| 角度 | 效果 |
|------|------|
| 平拍 | 客观、平等 |
| 高角度/俯拍 | 弱化、压迫 |
| 低角度/仰拍 | 权威、力量 |
| 倾斜角度 | 不安、紧张 |
| 过肩角度 | 对话、主观视角 |
| 航拍 | 宏观、壮丽 |
| 俯视角度 | 全知视角 |

### 镜头类型
- **干净的单人镜头**：聚焦单一角色
- **双人镜头**：对话、关系
- **三人镜头**：群体互动
- **群像镜头**：多人场景

### 色调
| 色调 | 情绪 |

## 声音设计

### 人声
```
"角色说话的内容" + 情绪 + 语调 + 语速 + 音色 + 口音
```
示例：男人说道："好好学习，天天向上"，语气轻松，语速适中，声音清亮

### 音效
```
音源材质 + 行为 + 环境音
```
示例：玻璃小球从桌面掉在木质地面上，发出"砰"的声音，室内安静环境

### 背景音乐
```
背景音乐/配乐 + 风格
```
示例：雨夜阴森走廊，配有悬疑风格背景音乐

## 使用约定

- 除代码和专有名词外，使用中文交流
- 提示词使用英文或中文，视模型特性选择
- 保持角色和场景的视觉一致性
- 遵循电影叙事逻辑进行创作
- 先构建完整的故事框架，再细化分镜
