## 关键节奏把控

```
铺垫(3s) → 出场(4s) → 升格美感(6s) → 经典台词(8s)
    ↓
BGM骤停 → 反转(无奈表情) → 捂耳朵 → "别听" → 定格
```

**节奏要点**：
1. 前半段要足够帅、足够燃、足够史诗
2. BGM骤停是关键反转点
3. 后半段要足够日常、足够无奈
4. 反差越大，喜剧效果越强
