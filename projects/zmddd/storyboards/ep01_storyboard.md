## 详细分镜

### 镜号 1-1｜定场镜头
```
景别：全景
运镜：缓慢上摇
时长：3秒
画面：丹青峰远景，云雾缭绕，山峰若隐若现
声音：山林环境音 + BGM轻起
```

**提示词**：
```
全景，白天，日光柔和，
丹青峰远景，云雾缭绕中的仙山，
青翠山林，石阶隐约可见，
仙气飘飘，古风仙侠，
竖屏构图，山峰居中，
cinematic establishing shot
```

---

### 镜号 1-2｜主角出场
```
景别：中景
运镜：跟随
时长：3秒
画面：路冬梨背着小秋拾级而上，青衫飘动
声音：脚步声
```

**提示词**：
```
中景，白天，自然日光，
古风少女背着小女孩走上石阶山道，
青色长衫，腰佩长剑，身姿高挑，
两侧山林，阳光斑驳，
跟随运镜，轻微晃动，
ancient Chinese xianxia style
```

---

### 镜号 1-5｜光影美感
```
景别：近景
运镜：缓慢推进
时长：3秒
画面：路冬梨侧脸特写，阳光洒在娇嫩脸庞
声音：对白
```

**提示词**：
```
近景，白天，侧逆光，暖色调，
古风少女侧脸，清丽脱俗，
阳光洒在脸上，光影柔美，
表情沉静带微笑，
镜头缓慢推进，
cinematic portrait, beautiful lighting
```

---

### 镜号 1-9｜喜剧点

