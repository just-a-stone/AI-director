## 场景一：丹青峰山道

### 基本信息

| 项目 | 设定 |
|------|------|
| 名称 | 丹青峰山道 |
| 类型 | 外景 |
| 时间 | 白天（上午，阳光明媚） |
| 用途 | 第1、2集主场景 |

### 环境描述

蜿蜒的石阶山道，两侧是青翠的山林。
偶有云雾从山间飘过，增添仙气。
阳光透过树叶洒下斑驳光影。
远处可见山峰若隐若现。

### 视觉设定

| 维度 | 设定 |
|------|------|
| 整体风格 | 古风仙侠，自然山林 |
| 色彩基调 | 青绿为主，阳光金色点缀 |
| 光线氛围 | 日光，柔和，穿透树林的光线 |
| 氛围 | 清幽、仙气、生机 |

### 关键道具/元素
- 石阶（古朴，有青苔）
- 山林（竹林或松林）
- 云雾（点缀仙气）
- 阳光光斑

### AI生成提示词

```
古风仙侠山道场景，
蜿蜒的石阶向山上延伸，
两侧青翠山林，竹林或松林，
云雾缭绕，仙气飘飘，
阳光透过树叶洒下斑驳光影，
远处山峰若隐若现，
ancient Chinese xianxia style, mountain path,
stone steps, bamboo forest, misty atmosphere,
natural daylight, cinematic, high quality
```

## 场景二：丹青峰峰顶

### 基本信息

| 项目 | 设定 |
|------|------|
| 名称 | 丹青峰峰顶 |
| 类型 | 外景 |
| 时间 | 白天（接近正午，光线充足） |
| 用途 | 第3集主场景（名场面） |

### 环境描述

山峰顶端，视野开阔。
脚下是翻涌的云海，远处群山连绵。
寒风猎猎，衣袂飘飘。
是一个适合"装逼"的绝佳位置。

### 视觉设定

| 维度 | 设定 |
|------|------|
| 整体风格 | 古风仙侠，大气磅礴 |
| 色彩基调 | 青白为主，云海银白 |
| 光线氛围 | 逆光/侧逆光，勾勒人物轮廓 |
| 氛围 | 恢弘、仙气、史诗感 |

### 关键道具/元素
- 云海（翻涌、壮观）
- 远山（层叠、若隐若现）
- 风（让衣衫飘动）
- 可能有简单的石台/栏杆

### AI生成提示词

```
古风仙侠山峰顶端场景，
脚下云海翻涌，远处群山连绵，
逆光照射，光芒万丈，
寒风猎猎，仙气磅礴，
壮观大气，史诗感，
ancient Chinese xianxia style, mountain peak,
sea of clouds, dramatic backlighting,
epic atmosphere, cinematic wide shot,
high quality, 8k
```

### 特别说明

这是**路朝歌出场的名场面**，场景要足够大气，
配合逆光和云海，营造"高手出场"的视觉效果，
与后续的中二台词形成反差。

## 场景三：墨门室内（闪回用）

### 基本信息

| 项目 | 设定 |
|------|------|
| 名称 | 墨门室内 |
| 类型 | 内景 |
| 时间 | 夜晚/白天（闪回） |
| 用途 | 第2集闪回场景 |

### 环境描述

简朴的古风室内，可能是书房或居室。
有窗户，可以看到外面的夜空/远景。
陈设简单但有文人气息。

### 视觉设定

| 维度 | 设定 |
|------|------|
| 整体风格 | 古风仙侠，简朴文雅 |
| 色彩基调 | 暖木色为主，闪回可降饱和度 |
| 光线氛围 | 烛光（夜）/ 窗边日光（日） |
| 氛围 | 安静、私密、回忆感 |

### 关键道具/元素
- 窗户（可看外景）
- 书案/茶几
- 烛台（夜景）
- 简单陈设

### AI生成提示词

**夜景版本（45度角仰望）**
```
古风仙侠室内场景，夜晚，

窗户打开，可见夜空星辰，
烛光摇曳，氛围安静，
适合回忆/闪回画面，略微降低饱和度，
ancient Chinese style interior, night,
wooden study room, candlelight,
window with night sky view,
nostalgic atmosphere, cinematic
```

**日景版本（窗边喝茶）**
```
古风仙侠室内场景，白天，
窗边位置，阳光洒入，
茶几上放着茶具，
简约文雅，仙门弟子居所，
ancient Chinese style interior, daytime,
window seat, natural light,
tea set, elegant and simple
```

## 竖屏构图注意

由于是9:16竖屏短剧，场景构图需注意：

1. **山道场景**：人物居中偏下，上方留山林
2. **峰顶场景**：人物居中，背景云海要够壮观
3. **室内场景**：可以更聚焦人物，背景点缀即可

避免过于横向展开的大全景，优先保证人物清晰可见。
