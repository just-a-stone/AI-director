# 《掌门大大》故事大纲

> 基于小说第一章「不听不听」改编

---

## 一、项目定位

| 维度 | 设定 |
|------|------|
| 类型 | 真人古风仙侠 · 轻喜剧 |
| 调性 | 沙雕搞笑 + 仙侠美学 |
| 时长 | 单集 1-2 分钟 |
| 集数 | 本章预计 3-5 集 |
| 平台 | 抖音/快手竖屏短剧 |
| 画幅 | 9:16 竖屏 |

---

## 二、核心卖点

**人设反差**：
- 外表：仙气飘飘、俊美无双的剑修掌门
- 内在：重度中二病穿越者，自认主角模板

**喜剧张力**：
- 路朝歌的中二发言 vs 路冬梨的无奈吐槽
- "天不生我路朝歌，天玄万古如长夜" —— 经典名场面

**情感内核**：
- 妹妹对哥哥的无奈与守护
- 表面嫌弃，实则深爱

---

## 三、世界观速览

| 要素 | 设定 |
|------|------|
| 世界 | 天玄界 |
| 地域 | 青州（剑修之州，七成修士为剑修） |
| 宗门 | 剑宗 → 墨门（附属宗门之一） |
| 地点 | 丹青峰 |

---

## 四、主要人物

### 路朝歌（男主）
- **身份**：墨门掌门
- **年龄**：22-23岁
- **外形**：俊美无双，青衫飘逸
- **性格**：极度自信（中二），自认穿越者/主角模板
- **口头禅**："天不生我路朝歌，天玄万古如长夜"
- **人设关键词**：拽、帅、中二、反差萌

### 路冬梨（女主）
- **身份**：墨门弟子，路朝歌的妹妹
- **年龄**：16-17岁
- **外形**：身姿高挑，双腿修长，娇嫩粉颈
- **性格**：早熟稳重，凡事谋定后动
- **人设关键词**：稳健、无奈、吐槽担当、护兄狂魔

### 小秋（配角）
- **身份**：路冬梨新收的徒弟
- **年龄**：约6-8岁
- **外形**：圆脸肉嘟嘟，大眼水灵
- **性格**：乖巧天真，童言无忌
- **作用**：观众视角代入、引出信息、萌点担当

---

## 五、第一章剧情拆解

### 叙事结构：起承转合

**起｜上山路上**
- 路冬梨背着小秋上丹青峰
- 介绍宗规、宗门情况
- 铺垫"掌门师伯很拽"

**承｜铺垫人设**
- 路冬梨回忆哥哥的种种中二言行
- "不过是一款游戏""主角模板"
- 父母去世后反而更自信了

**转｜名场面登场**
- 到达山顶，路朝歌出场
- 青衫负手，迎风望天
- 说出经典台词

**合｜点题收尾**
- 路冬梨捂住小秋耳朵
- "别听"
- 呼应章节名「不听不听」

## 八、改编要点

1. **保留核心**：路朝歌的中二人设是最大卖点，必须保留
2. **强化反差**：外形要足够帅，台词要足够中二
3. **精简叙述**：小说中的背景介绍通过画面/旁白带过
4. **增加节奏**：每集都要有笑点或钩子
5. **竖屏适配**：构图以人物为主，减少大全景
